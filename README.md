## Problem Description

Traditional food ordering methods often lead to:
- Order confusion
- Billing errors
- Delays in food preparation and delivery

The **Online Food Ordering System** is developed to simplify the complete food ordering process digitally. Customers can browse menus, place orders, make payments, and track delivery status in real time.

The system includes:
- Customer management
- Restaurant and menu handling
- Order processing
- Payment management
- Delivery tracking

This project demonstrates core **Object-Oriented Programming (OOP)** concepts in Java such as:
- Inheritance
- Encapsulation
- Abstraction
- Polymorphism
- Interfaces
